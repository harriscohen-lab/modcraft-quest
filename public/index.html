ModCraft Quest

Build Your First Minecraft Mod — Step by Step

A friendly, hands-on learning path for a super-smart 6th grader. We’ll set up your tools, practice real code, and ship a finished mod — while you track your progress and pick up right where you left off.

1) Setup: Chromebook or Ubuntu

You can use a Chromebook (with Linux enabled) or an Ubuntu laptop. Both options let you code using Java and run Minecraft Java Edition with the Fabric mod loader.

Chromebook (Linux)

  1. Open Settings → Developers → turn on Linux.
  2. Install Java 17 in the Linux terminal.
  3. Install VS Code (or a similar code editor).
  4. Download Minecraft Java Edition.
  5. Install the Fabric Loader for your Minecraft version.
Quick terminal commands 
sudo apt update
sudo apt install -y openjdk-17-jdk
java -version

Ubuntu Laptop

  1. Install Java 17 from the terminal.
  2. Install VS Code or IntelliJ IDEA.
  3. Download Minecraft Java Edition.
  4. Install Fabric Loader.
  5. Create a new “mods” folder inside your Minecraft folder.
Quick terminal commands 
sudo apt update
sudo apt install -y openjdk-17-jdk
java -version
Why Fabric? Fabric is lightweight and friendly for beginner modders, and it’s popular in the modding community.

2) Weekly Program (8 Weeks)

Check off each week as you go. Your progress is saved in this browser, so you can come back later.

0% complete
  • Watch: “What is Fabric Modding?” (YouTube search)
  • Do: Write a tiny Java program that prints “Hello, Modder!”
  • Goal: Understand files, folders, and compiling.

How to do Week 1

  1. Create a folder named java-practice.
  2. Open your editor and create a file named HelloModder.java.
  3. Paste this code: public class HelloModder { public static void main(String[] args) { System.out.println("Hello, Modder!"); } }
  4. Open a terminal in the folder and run: javac HelloModder.java.
  5. Then run: java HelloModder.
  6. Take a screenshot of the output for your journal.
  • Watch: “Resource Packs Explained” (YouTube search)
  • Do: Create a resource pack that changes a block texture.
  • Goal: Learn how Minecraft loads assets.

How to do Week 2

  1. Make a new folder called MyFirstPack.
  2. Create pack.mcmeta with a description and version.
  3. Inside assets/minecraft/textures/block, add a 16×16 PNG named stone.png.
  4. Zip the folder and move the zip to Minecraft’s resourcepacks folder.
  5. Enable it in Minecraft → Options → Resource Packs.
  6. Take a screenshot of the new texture in game.
  • Watch: “Fabric Setup Guide” (YouTube search)
  • Do: Generate a Fabric mod project and run it in the dev client.
  • Goal: Launch Minecraft from code.

How to do Week 3

  1. Go to the Fabric example mod generator in your browser.
  2. Pick your Minecraft version and mod name.
  3. Download the zip and unzip it to a workspace folder.
  4. Open the folder in your editor.
  5. In a terminal, run: ./gradlew runClient.
  6. When Minecraft opens, check the “Mods” list for your mod name.
  • Watch: “Creating Items in Fabric” (YouTube search)
  • Do: Add a “Glow Berry Wand” item.
  • Goal: Register items and textures.

How to do Week 4

  1. Open your mod initializer Java file.
  2. Create a new Item and register it with a unique ID.
  3. Add the item to a creative tab so it shows up in-game.
  4. Create an item JSON model file in assets/yourmod/models/item.
  5. Restart the dev client and find your wand in the creative menu.
  • Watch: “Adding Item Abilities” (YouTube search)
  • Do: Make the wand spawn glowing particles.
  • Goal: Trigger events in code.

How to do Week 5

  1. Create a custom item class that extends Item.
  2. Override the use method to detect right-click.
  3. Spawn particles at the player’s location.
  4. Play a sound effect so it feels magical.
  5. Test in the dev client and adjust the effect strength.
  • Watch: “Blocks in Fabric” (YouTube search)
  • Do: Create a “Moonstone Block” with a new texture.
  • Goal: Register blocks and block items.

How to do Week 6

  1. Create a new Block and register it.
  2. Create a block item so players can hold it.
  3. Add a blockstate JSON file and block model JSON file.
  4. Draw a 16×16 texture and save it in the block textures folder.
  5. Run the game and place the Moonstone Block.
  • Do: Combine your wand + moonstone into a mini adventure.
  • Goal: Write a short README and mod description.

How to do Week 7

  1. Write a short story: “Find Moonstone to power the wand.”
  2. Define 3 tasks in-game (collect, craft, test).
  3. Add a crafting recipe JSON for your wand or block.
  4. Write a README with install steps and screenshots.
  5. Share the README with a friend or parent for feedback.
  • Do: Build the mod .jar file and test it in Minecraft.
  • Goal: Share the mod with a friend on Ubuntu/Chromebook.

How to do Week 8

  1. Run ./gradlew build to create the jar.
  2. Copy the jar into the mods folder.
  3. Launch Minecraft with Fabric Loader.
  4. Verify the item + block work in survival mode.
  5. Zip the jar and send it to a friend using the same version.

3) Hands-On Build: The Glow Berry Wand Mod

This is your first complete mod. You’ll create a custom item, customize its texture, and give it a cool ability.

Step A: Create the mod

  • Use the Fabric project generator (Gradle).
  • Pick a mod ID like glow_berry_wand.
  • Run the dev client to check everything works.

Step B: Register the item

  • Add a new Item in your mod initializer.
  • Give it a creative tab.
  • Make sure the item appears in-game.

Step C: Add the texture

  • Draw a 16×16 PNG in any art tool.
  • Save it in the textures folder.
  • Reload the game to see it.

Step D: Add ability code

  • Use Fabric events to detect right-click.
  • Spawn particles and play a sound.
  • Optionally give the player a speed boost.

✅ By the end, you have a full mod with code, assets, and a gameplay effect.

4) Packaging & Installing Your Mod

Nintendo Switch versions of Minecraft do not support mods unless the console is modified (which isn’t recommended). So we’ll install the mod on Ubuntu/Chromebook instead.

Build the mod file

  1. Open the project folder in a terminal.
  2. Run ./gradlew build.
  3. Find the .jar file in build/libs.

Install on Ubuntu / Chromebook

  1. Open the Minecraft folder.
  2. Drop the mod .jar into the mods folder.
  3. Start Minecraft with Fabric Loader.
Want to share? Zip the build/libs file and send it to a friend who also uses Fabric.
